# Newaita-Shadow-Dark

Hybrid icon theme combining:

- **Base:** [Newaita-reborn-dark](https://github.com/cybernix/Newaita-reborn) — actions, devices, emblems, mimetypes, places, panel/status, and categories
- **Application icons:** [Shadow Dark](https://github.com/L4ki/Shadow) — all app icons from the Shadow Dark theme

## Install

```bash
./install.sh
```

Or manually:

```bash
ln -sfn "$(pwd)" ~/.local/share/icons/Newaita-Shadow-Dark
```

Then select **Newaita-Shadow-Dark** in your desktop environment's icon theme settings.

## Rebuild

If you update either source theme, rerun:

```bash
./build.sh
./install.sh
```
